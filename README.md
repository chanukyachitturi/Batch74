# ☕ Core Java Practice Programs

A collection of Core Java programs created while learning and practicing Java fundamentals.

## 📚 Topics Covered

### 1. Classes and Objects
- Object creation
- Instance variables
- Static variables
- Method invocation

### 2. Constructors
- Parameterized constructors
- Constructor execution flow
- Object initialization

### 3. Static Members
- Static variables
- Static methods
- Static object references

### 4. Class Loading
- Class.forName()
- Dynamic class loading
- ClassLoader concepts

### 5. Garbage Collection
- Object eligibility for GC
- finalize() method
- System.gc()

### 6. Exception & Error Demonstrations
- StackOverflowError
- Recursive object creation
- Recursive method calls

### 7. Java Memory Concepts
- Heap Memory
- Stack Memory
- Object References
- Static Memory Allocation
